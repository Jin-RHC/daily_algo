### 1. 여행 경로

헷갈리는 요소가 너무 많았다. ~~누가 여행을 이렇게 가;~~  어쩐지 시간이 많이 걸렸다. 핵심은 모든 도시를 방문해야 하며, 티켓은 겹치는 것이 있을 수도 있지만 일단 모든 항공권을 사용해야 한다는 것. 또, 경로가 여러 개면 알파벳 순서가 앞서는 경로를 반환해야 한다는 것이다. 처음부터 항공권을 알파벳 순으로 나열하여 풀었다.
